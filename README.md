# C Calculator

A simple calculator program written in C using switch statements.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## How to Compile

```bash
gcc Calc.c -o calculator
````

## How to Run

### Windows

```bash
calculator.exe
```

### Linux/macOS

```bash
./calculator
```
